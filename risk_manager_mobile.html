Коэффициентный анализ

ICR = EBIT / Проц. расходы · D/E = ЗК / СК · К_тл = ТА / ТО · К_абс = Деньги / ТО. Нормативы: Ковалёв В.В. (2002), Brigham & Houston (2019).

Z'-модель Альтмана (1968)

Для непубличных компаний. Z' <1,23 — зона банкротства; 1,23–2,90 — серая; >2,90 — безопасная.

Параметрический VaR (Jorion, 2007)

VaR₉₅ = μ + 1,645·σ; VaR₉₉ = μ + 2,326·σ. При одном наблюдении используется сценарный диапазон.

NPV хеджирования

Дисконтированная разность экономии и затрат за 3 года по ставке WACC. Оптимальный коэф. хеджирования h* ≈ 0,5 (Hull, 2022).
